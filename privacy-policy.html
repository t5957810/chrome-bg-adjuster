ğŸ“‹ Overview

This Privacy Policy describes how the "Background Color Adjuster" Chrome Extension (hereinafter "the Extension") handles user information.

The Extension is maintained by an individual developer and does not collect, store, or transmit any personal user data in any form.

Last updated: February 24, 2026

ğŸ”’ Data Collection

The Extension does not collect any of the following:

  • Personally identifiable information (name, email, IP address, etc.)
  • Browsing history or webpage content
  • Usage behavior analytics
  • Device information
  • Any form of telemetry or diagnostic data

ğŸ’¾ Local Storage

The Extension only uses chrome.storage.local to store the following settings on the user's local device:

  • Per-site enabled/disabled state
  • Selected color mode (auto-invert / custom colors)
  • Custom background color, text color, and link color preferences

This data never leaves the user's device and is never uploaded to any server or third-party service.

ğŸŒ Permissions Justification

  • activeTab: Only used when the user clicks the extension icon to read the current tab's origin, in order to display the current setting status.
  • storage: Saves the user's color preferences locally so they are automatically applied on future visits to the same site.
  • scripting: Injects CSS styles into the page to adjust background and text colors visually. Does not read page content.
  • All URLs access: Since users may wish to use this feature on any website, it is not possible to pre-specify domains. The extension is disabled for all sites by default and must be manually enabled per site by the user.

ğŸ¤ Third-Party Services

The Extension does not use any third-party analytics, advertising services, or tracking code such as Google Analytics or Firebase.

ğŸ‘¶ Children's Privacy

The Extension is not directed at children under 13 and does not collect any user data, so there are no children's data protection concerns.
